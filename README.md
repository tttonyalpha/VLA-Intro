# Hands-on Vision-Language-Action (VLA) models

<img width="3038" height="1408" alt="prev" src="https://github.com/user-attachments/assets/770911d0-2d3e-4368-9554-f00a130c02aa" />


VLA models connect **visual perception**, **language understanding**, and **robot actions** in a single system. In this seminar, we first discuss the core ideas behind VLA models and then move to practical tools for training and deploying them in robotics workflows.

---

## Introduction to VLA Models

In the first part, we cover the fundamentals of VLA models:

- what VLA models are
- how they work
- what data they use
- how they are trained
- common architectural design choices
- a practical example based on **SmolVLA**
- how to train VLA using RL
